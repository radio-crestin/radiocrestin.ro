# radiocrestin.ro

## Dev Server
- Port: **3005** (`npm run dev` runs on http://localhost:3005)

## Stack
- Next.js 15 (pages router, static generation)
- TypeScript, SCSS Modules, Zustand
- API: https://api.radiocrestin.ro/api/v1/stations

## Project Structure
- `src/pages/` — Next.js pages
- `src/common/components/` — Shared components
- `src/common/models/` — TypeScript interfaces
- `src/common/services/` — API calls
- `src/common/store/` — Zustand stores
- `public/icons/` — SVG icon components
